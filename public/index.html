What is FiveM IP Finder?

FiveM IP Finder is a free online tool that helps you find the IP address of any FiveM server using its CFX.re URL or server ID. Perfect for GTA V FiveM players who want to connect directly to servers.

How to use FiveM IP Finder:

  1. Copy the CFX.re URL from FiveM (e.g., cfx.re/join/abc123)
  2. Paste it into the search box above
  3. Click "Find IP" button
  4. Get the server IP address, port, and player information instantly
